## Get started

1. `npm install`
2. `npm run dev`
3. Go to `http://localhost:3000/`

## APIs used
1. Openweather API: https://openweathermap.org/api
2. Gfycat API: https://developers.gfycat.com/api/

## Setup environment for full experience locally

1. Set Openweather API key (export WEATHER_API_KEY=XXXXXXXX)
2. Set Gfycat Id (export GFYCAT_ID=XXXXXXXXXX)
3. Set Gfycat Secret (export GFYCAT_SECRET=XXXXXXXXXX)
